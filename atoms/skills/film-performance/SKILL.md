# Film Performance

表演节奏先于分镜——先想清楚人物在这场戏里怎么行动，分镜才有切换的依据。

---

## 模式一：定调（项目开始时用一次）

输入：节拍表和人物小传。

### 表演风格基准

定义这部影片的整体表演幅度：

- **克制内敛**：情绪藏在动作里，不外化。沉默比台词多，微表情比大动作多。（参考：侯孝贤、是枝裕和）
- **外化爆发**：情绪直接表达，动作幅度大，冲突激烈。（参考：张艺谋、科波拉）
- **混合**：日常戏克制，情绪高潮外化，形成对比张力

同时定义：
- 沉默的使用策略：沉默是常态还是例外？沉默出现时意味着什么？
- 肢体语言的密度：动作频繁还是克制？

输出示例：
```
表演风格：克制内敛为主，情绪藏于细节
沉默策略：沉默是常态，开口即意味着重要
肢体密度：动作极度克制，静止是默认状态
```

### 全片张力曲线

对照节拍表，标注情绪积累和释放的关键节点：

- 积累段：哪些节拍是情绪压抑积累的阶段（表演幅度收小）
- 爆发点：哪些节拍是情绪释放的时机（表演幅度放开）
- 余韵段：爆发后的沉淀阶段（表演回归克制）

输出示例：
```
节拍 1-8（积累）：表演幅度 20%，克制，压抑
节拍 9（中点）：幅度升至 40%，开始松动
节拍 11（一无所有）：爆发点，幅度 80%
节拍 12-13（余韵）：幅度回落至 30%，沉淀
节拍 14（结局）：幅度 60%，释放但不失控
```

## 模式二：场景执行（每次分镜时调用）

输入：当前场景的叙事目的 + 全片定调基准。

### 行动节拍设计

逐步骤设计这场戏的人物行动：

**第一步：确认情绪状态**
- 当前场景处于张力曲线的哪个位置？
- 人物此刻的内心状态是什么（来自人物小传）？

**第二步：设计行动节拍**

行动节拍是镜头切换的依据：

```
节拍 1：[人物] [动作/静止] → [情绪变化] → 切换时机
节拍 2：[人物] [动作/静止] → [情绪变化] → 切换时机
```

- 动作：具体的身体行为（站起、转身、放下杯子、沉默）
- 切换时机：动作完成前切 / 完成后切 / 动作中切（影响节奏感）

**第三步：对照基准检查**
- 这场戏的表演幅度是否符合全片风格基准？
- 如果偏离，叙事理由是什么？

### 输出格式

```
场景：[场景名] | 节拍位置：[节拍编号] | 情绪状态：[描述]

行动节拍：
1. [人物]：[动作] — 切换时机：[前/中/后]
2. [人物]：[动作] — 切换时机：[前/中/后]
...

表演幅度：[百分比] | 基准符合：[是/否，偏离原因]
```
