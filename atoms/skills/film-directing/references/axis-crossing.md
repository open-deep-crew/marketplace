# 越轴处理方案

越轴（跨越 180° 轴线）会让观众产生空间迷失感。以下是合法的越轴方式。

## 何时需要越轴

- 叙事需要改变人物的空间关系（如从对立变为并肩）
- 追逐戏中方向需要反转
- 场景情绪需要通过空间颠覆来表达

## 三种合法越轴方式

### 方式一：中性镜头过渡

在越轴前插入一个正面镜头（摄影机在轴线上）：

```
镜头 A（轴线左侧）→ 中性镜头（正面，轴线上）→ 镜头 B（轴线右侧）
```

- 中性镜头通常是人物的正面特写或正面中景
- 观众在中性镜头中"重置"空间感
- 最常用、最安全的越轴方式

### 方式二：移动镜头越轴

摄影机在运动中越过轴线：

```
镜头 A（轴线左侧）→ 摄影机移动越过轴线 → 落点在轴线右侧
```

- 摄影机必须在同一个连续镜头内完成越轴
- 观众跟随摄影机运动，自然接受新的空间关系
- 适合追逐戏、人物移动改变方向的场景

### 方式三：空镜过渡

在越轴前插入一个无人物的空间镜头：

```
镜头 A（轴线左侧）→ 空镜（环境、道具）→ 镜头 B（轴线右侧）
```

- 空镜打断了观众对空间连续性的期待
- 重新建立后，新的轴线被接受
- 适合时间跳跃或情绪转换的场景

## 越轴的叙事意义

有时越轴是刻意的叙事手段：

- **戈达尔式跳切**：刻意打破连续性，制造疏离感
- **情绪颠覆**：通过空间混乱表达人物心理混乱
- **权力关系转变**：越轴后新的空间关系暗示权力易位

使用前确认：这是叙事需要，还是失误？

## 检查清单

在完成分镜设计后，逐镜检查：

- [ ] 所有对话镜头是否在同一轴线侧？
- [ ] 人物视线方向是否在相邻镜头中保持相对？
- [ ] 如有越轴，是否安排了过渡镜头？
- [ ] 越轴是否有叙事理由？
