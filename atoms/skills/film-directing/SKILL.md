# Film Directing

场景级分镜执行。使用前提：全片视听风格（`/film-visual-language`）、表演基准（`/film-performance`）、美术方案（`/film-art-direction`）已完成定调。

详细越轴处理方案见 [references/axis-crossing.md](references/axis-crossing.md)。

## 分镜工作流程

### 第零步：确认钩子

在设计任何镜头之前，先明确这场戏的钩子：

- **进入钩子**：第一个镜头用什么视听手段打出编剧设计的钩子？（细节特写先行 / 声音悬念 / 反常构图 / 动作截断）
- **离开钩子强化**：最后一个镜头如何放大叙事留下的未解张力？（定格在关键道具 / 人物表情截断 / 声音延续画面结束）

钩子镜头不一定是主镜头，可以是一个独立的开场细节或收尾特写。

### 第一步：估算镜头数量

根据场景时长和情绪类型确定镜头数量范围：

| 场景类型 | 镜头数量 | 平均时长/镜 |
|---------|---------|-----------|
| 对话场景（两人） | 6-8 个 | 8-12 秒 |
| 情绪独白 | 3-5 个 | 10-20 秒 |
| 动作场景 | 10-20 个 | 2-5 秒 |
| 情绪沉淀 | 2-4 个 | 15-30 秒 |

原则：够用就停，不为镜头数量而加镜头。

### 第二步：确定轴线

- 画场景平面图，标注人物位置
- 确定动作轴线或视线轴线
- 所有机位必须在轴线同侧
- 需要越轴时参见 [references/axis-crossing.md](references/axis-crossing.md)

### 第三步：按顺序设计镜头

**1. 主镜头（Master Shot）**
- 全景或中景，覆盖整场戏
- 建立空间关系和人物位置
- 这是场景的叙事基础

**2. 分切镜头（情绪强化点）**
- 在情绪节点处加近景或特写
- 对应 `film-performance` 的行动节拍切换时机
- 每个情绪强化点 1 个镜头，不重复

**3. 反应镜头**
- 捕捉对方人物的反应
- 通常是近景或特写
- 视线匹配：A 看右，B 必须看左

### 第四步：检查衔接规则

- **30° 规则**：相邻镜头机位夹角 ≥30°
- **视线匹配**：人物看哪，下一镜头接被看的对象
- **位置匹配**：人物画面左右位置不颠倒
- **动作匹配**：动作跨镜头连贯不重复

### 第五步：输出完整分镜稿

每个镜头整合所有 skill 的结果，格式如下：

```
【镜号】
景别：[远/全/中/近/特写]
机位：[描述机位角度和位置]
镜头运动：[静止/推/拉/摇/移/跟/升/降]
时长预估：[秒]

叙事目的：[这个镜头让观众感受到什么]

人物行动：[动作描述 + 切换时机（前/中/后）]
（来自 film-performance）

视觉元素：
- 色调：[主色 + 情绪色彩状态]
- 光源：[来源 + 色温 + 方向]
- 关键道具/陈设：[具体描述]
（来自 film-art-direction）

视觉校验：[符合/偏离全片基准，偏离原因]
（来自 film-visual-language）
```

## 景别叙事逻辑

在全片景别策略约束下执行：

| 景别 | 叙事功能 | 使用时机 |
|------|---------|---------|
| 远景（ELS） | 建立地理位置 | 场景开头、时空转换 |
| 全景（LS） | 定人物在空间中的位置 | 主镜头 |
| 中景（MS） | 展示人物关系和互动 | 对话戏主体 |
| 近景（MCU） | 传递情绪状态 | 情感转折点 |
| 特写（CU） | 强调心理 / 关键细节 | 情绪高潮 |

铁律：景别变化必须有叙事触发条件，禁止无意义同景别硬切。

## 观众心理引导

- **距离 = 亲疏**：人物距离近 → 亲密/压迫；距离远 → 疏离/对立
- **客观镜头**：上帝视角，冷静叙事
- **主观镜头（POV）**：代入角色，强化情绪认同
- **画外空间**：用视线、动作、声音引导观众感知画外
