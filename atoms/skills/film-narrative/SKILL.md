# Film Narrative

基于 Blake Snyder 的 Save the Cat 方法论，从一个主题出发构建完整的叙事骨架。

## 工作流程

### 第一步：从主题提炼故事前提

用一句话写出故事前提（Logline）：

```
[主角] 因为 [催化事件]，必须 [目标]，否则 [代价]。
```

确认前提包含：
- 主角是谁（有缺陷的普通人）
- 世界发生了什么变化（催化事件）
- 主角的外部目标 vs 内心真正需要的东西
- 失败的代价

### 第二步：建立人物小传

每个核心角色输出：

```
姓名：
一句话定位：
外部欲望（Want）：表面上想要什么
内心需求（Need）：真正需要什么（通常与欲望冲突）
最大恐惧：
弧光起点：故事开始时的状态
弧光终点：故事结束时的变化
```

### 第三步：填写 Save the Cat 节拍表

按 15 个节拍顺序输出，每个节拍包含：事件描述（一句话）+ 情绪状态标注。

节拍定义见 [references/save-the-cat-beats.md](references/save-the-cat-beats.md)。

### 第四步：提炼核心场景清单

从节拍表中提取 5-8 个必须拍摄的关键场景：

```
场景编号：
对应节拍：
一句话描述：
情绪目标：（这场戏要让观众感受到什么）
人物关系：（谁和谁，什么关系状态）
进入钩子：（前10秒用什么抓住观众？悬念/冲突/反常/强烈视觉/声音先行）
离开钩子：（这场戏结束时留下什么未解决的张力，驱动观众看下一场）
```

## 输出格式

完成后输出三份文档：
1. **人物小传**（所有核心角色）
2. **节拍表**（15个节拍，含情绪标注）
3. **核心场景清单**（5-8个场景，含情绪目标）

这三份文档直接作为后续阶段的输入：
- 节拍表 → 视听语言规划
- 核心场景清单 → 分镜构思
- 人物小传 → 表演节奏控制
