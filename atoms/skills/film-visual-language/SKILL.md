# Film Visual Language

两个使用模式，服务于不同阶段。

---

## 模式一：定调（项目开始时用一次）

输入：叙事定调阶段输出的节拍表和核心场景清单。

目标：建立这部影片的视觉决策框架，供后续所有分镜参照。

### 景别变化逻辑

不是定"全片用什么景别"，而是定"景别变化的叙事触发条件"：

- 什么情绪状态驱动景别收紧（中景 → 近景 → 特写）？
- 什么叙事事件驱动景别放开（特写 → 中景 → 全景）？
- 特写的使用策略：克制还是频繁？克制意味着特写出现时有更大的重量。

输出示例：
```
景别收紧触发：人物内心冲突升级、情绪即将爆发
景别放开触发：时空转换、人物关系疏离、场景建立
特写策略：克制使用，仅在节拍 11（一无所有）和节拍 14（结局高潮）出现
```

### 色彩转变节点

不是定"全片色调"，而是定"色彩在哪些叙事节点发生转变，以及为什么"：

- 对照节拍表，标注哪些节拍需要色彩变化
- 每次转变说明叙事理由（不是"这里要暖色"，而是"主角找到希望，色温从冷转暖"）
- 基准色调：没有特殊叙事事件时，影片默认的色彩状态

输出示例：
```
基准色调：冷色、低饱和（疏离感）
节拍 6（进入新世界）：色温略升，饱和度提高——主角开始相信改变的可能
节拍 11（一无所有）：最冷、最低饱和，接近黑白
节拍 13（转折）：第一次出现暖色，小面积，暗示希望
```

### 节奏基准

定义这部影片的默认剪辑节奏，以及什么情况下打破它：

- 基准节奏：日常叙事段落的镜头时长和切换频率
- 加速触发：什么叙事事件驱动节奏加快（冲突升级、追逐、情绪爆发）
- 减速触发：什么叙事事件驱动节奏放慢（内心独白、情感沉淀、时间流逝）

### 摄影机运动基调（可选）

仅当导演有明确的运动美学主张时填写：

- 基准状态：这部影片的摄影机默认是静止还是运动的？
- 运动的叙事意图：摄影机开始运动意味着什么？（跟随人物的主观代入 / 推进制造压迫感 / 拉开制造疏离）
- 手持 vs 稳定：手持传递不稳定感，稳定传递冷静或压迫，全片倾向哪个？

## 模式二：分镜校验（每次做分镜时调用）

输入：当前场景的叙事目的 + 已建立的视觉决策框架。

对每个镜头的视觉决策逐项检查：

**景别检查**
- 这个景别选择的叙事理由是什么？
- 是否符合已定义的景别变化逻辑？
- 如果是偏离，偏离的理由是什么？（刻意打破规则需要更强的叙事理由）

**色彩检查**
- 当前场景处于节拍表的哪个位置？
- 对应的色彩状态是什么？
- 如果和基准不同，转变的叙事依据是什么？

**节奏检查**
- 当前场景的剪辑节奏是否符合基准？
- 如果加速或减速，触发条件是否成立？

**一致性检查**
- 这个镜头的视觉决策和前面已完成的分镜是否在同一部影片的语言体系里？
- 有没有无意识的风格漂移？
