# Film Art Direction

将情绪色彩方案落地为具体的视觉元素。输出必须细到可以直接翻译成 AI 生成模型的提示词参数。

## 工作流程

### 第一步：确认输入

- 当前场景处于节拍表的哪个位置？情绪状态是什么？
- `film-visual-language` 定调的色彩框架对这个节拍的要求是什么？
- 时代背景和地点是什么？

### 第二步：推导视觉元素

从情绪状态和色彩框架出发，逐项推导：

**场景陈设**
- 主要家具/建筑元素：颜色 + 材质 + 状态（新旧、整洁/破败）
- 墙面/地面：颜色 + 材质
- 整体空间感：开阔/逼仄、明亮/昏暗

**光源**
- 光的来源（窗户自然光/油灯/电灯/蜡烛）
- 光的色温（暖/冷/中性）
- 光的强度和方向（硬光/散射光、顺光/侧光/逆光）

**服装**
- 主要角色服装颜色 + 材质 + 状态
- 颜色是否和场景形成对比或融合（对比=突出人物，融合=人物被环境吞没）

**关键道具**
- 叙事相关道具的具体视觉描述（颜色、材质、状态）
- 道具在画面中的位置和重要程度

### 第三步：输出检查

输出前确认每一项描述：
- 是具体的物件描述，不是抽象风格词（"深褐色旧木桌"而不是"复古风格"）
- 颜色有具体色调（"青灰色"而不是"冷色"）
- 材质有质感描述（"粗麻布"而不是"布料"）
- 可以直接被 prompt-alchemist 翻译成提示词参数

## 输出格式

```
场景：[场景名] | 节拍：[节拍编号] | 情绪：[情绪状态]

陈设：
- [元素]：[颜色] + [材质] + [状态]

光源：[来源] + [色温] + [强度/方向]

服装：
- [角色]：[颜色] + [材质] + [状态]

关键道具：
- [道具]：[颜色] + [材质] + [叙事作用]

主色：[颜色]
辅色：[颜色]
禁用色：[颜色]（与情绪冲突的颜色）
```
